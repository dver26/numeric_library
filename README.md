Aquest codi integra numèricament segons el mètode de Trapezis i Simpson, en simple i en compost. 
Pots escollir fins quants sub-intervals podem dividir l'interval original i fins i escollir una tolerància per quan no volguem 
que es repeteixin els resultats molt pròxims entre ells.

Per executar:
$ python main.py

Només es requereix la llibreria NumPy
